## String Pool
–ù–µ–∏–∑–º–µ–Ω—è–µ–º–æ—Å—Ç—å —Å—Ç—Ä–æ–∫ –ø–æ–∑–≤–æ–ª–∏–ª–æ –∫ –∫–ª–∞—Å—Å—É String –ø—Ä–∏–º–µ–Ω–∏—Ç—å –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏—é –≤ –≤–∏–¥–µ String Pool.
–ü–æ–ª–µ–∑–Ω–æ–µ –æ–±—Å—É–∂–¥–µ–Ω–∏–µ –Ω–∞ stackoverflow: ["Where does Java's String constant pool live, the heap or the stack?"](https://stackoverflow.com/questions/4918399/where-does-javas-string-constant-pool-live-the-heap-or-the-stack).

–ß—Ç–æ –∂–µ –¥–∞—ë—Ç –Ω–∞–º String pool? –£ —ç—Ç–æ–≥–æ –µ—Å—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –ø—Ä–µ–∏–º—É—â–µ—Å—Ç–≤:
- –ù–µ –±—É–¥—É—Ç —Å–æ–∑–¥–∞–≤–∞—Ç—å—Å—è –æ–¥–Ω–æ—Ç–∏–ø–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã
- –°—Ä–∞–≤–Ω–µ–Ω–∏–µ –ø–æ —Å—Å—ã–ª–∫–µ –±—ã—Å—Ç—Ä–µ–µ, —á–µ–º –ø–æ—Å–∏–º–≤–æ–ª—å–Ω–æ–µ —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ —á–µ—Ä–µ–∑ equals

–ù–æ –∑–∞ –ø–ª—é—Å–∞–º–∏ –≤—Å–µ–≥–¥–∞ —Å—Ç–æ—è—Ç –º–∏–Ω—É—Å—ã. –ï—Å–ª–∏ String —Å–æ–∑–¥–∞–≤–∞–ª—Å—è —á–µ—Ä–µ–∑ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä –∫–ª–∞—Å—Å–∞ String, —Ç–æ –±—É–¥–µ—Ç —Å–æ–∑–¥–∞–Ω –Ω–æ–≤—ã–π –æ–±—ä–µ–∫—Ç, –∫–æ—Ç–æ—Ä—ã–π –Ω–µ –±—É–¥–µ—Ç –ø–æ–º–µ—â—ë–Ω –≤ String pool, –±–µ–∑ —è–≤–Ω–æ–≥–æ –Ω–∞ —Ç–æ —É–∫–∞–∑–∞–Ω–∏—è. –í–æ—Ç –ø—Ä–∏–º–µ—Ä:
